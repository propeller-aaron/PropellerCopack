SEO analysis — Propeller Copack website

Scan date: reflects the repository as of the last regeneration. Scope: 15 public marketing URLs (index.html plus 14 section folders). This file (seo.html) is internal-only and excluded from that count.

What improved (since earlier baseline)

  • Homepage (/): Branded <title> (“Propeller Co-Pack | Powder Blending, Packaging & Fulfillment”), a filled meta description, and a substantive meta keywords list.
  • Canonical URLs: <link rel="canonical" href="https://propellercopack.com/..."> is present on the homepage and on inner pages checked (e.g. /blending/, /fda/, etc.).
  • Homepage service checklist: Links now target real paths (./fda/, ./testing/, ./40000-sq-ft/, …), not only ./.
  • XML sitemap: sitemap.xml lists the homepage and service URLs under https://propellercopack.com/.

Executive summary

The site remains a static multi-page build with folder URLs, lazy-loaded images, and WebP via <picture>. Crawl signals are stronger on the homepage than on inner pages. The largest remaining gaps are empty meta descriptions on all inner pages, thin generic <title> tags on those pages, no robots.txt (so the sitemap is not declared to crawlers), missing Open Graph / Twitter tags, repeated hero blocks on inner URLs (near-duplicate patterns), weak or wrong image alt text and sitemap image titles, and minor copy issues on the homepage.

Critical issues (prioritize)

1. 40000-sq-ft document title does not match content

The page <title> is still “3900 Sq. Ft.” while the on-page heading describes a 40,000 sq. ft. facility. Fix the title (and ensure SERP alignment with canonical + H1).

2. Packaging design page: title, heading, image alt, and sitemap

<title> is “Package Design”, the H1 is “Packaging Design”, the hero image alt is still “Blending”, and sitemap.xml uses <image:title>Blending</image:title> for the packaging-design images. Align naming everywhere for topical relevance.

3. sitemap.xml image metadata quality

Beyond packaging-design, many inner URLs’ image entries reuse “Blending” as image:title for unrelated topics (FDA, testing, 3PL, etc.). That dilutes relevance for image search and internal consistency; regenerate titles per page topic.

Technical SEO

  • sitemap.xml: Present and lists main URLs; fix image titles as above.
  • Canonicals: Implemented on sampled pages — good for duplicate URL handling.
  • JSON-LD: Not observed — consider Organization / LocalBusiness using footer NAP.

Images and media

  • Strengths: Dimensions on many images, lazy loading, WebP sources.
  • Weaknesses: Some alt values mirror URL-encoded filenames; footer logo uses encoded-style alt in places; align alt with visible intent.

Internal linking

Homepage checklist links deep-link to service folders correctly. Continue cross-linking related services where it helps users (without stuffing keywords).

Measurement

Analytics placeholders in page heads may still be empty — confirm Google Search Console property for propellercopack.com and connect analytics.

Prioritized recommendations

  1. Fix 40000-sq-ft <title> to match 40,000 sq. ft. messaging.
  2. Align packaging-design: <title>, H1, image alt, and sitemap image:title.
  3. Rewrite sitemap.xml image titles for every URL (replace incorrect “Blending” duplicates).
  4. Add unique meta description and stronger <title> on each inner page.
  5. Add robots.txt with sitemap declaration.
  6. Add lang="en" on all marketing <html> elements.
  7. Correct homepage “Scahets” → “Sachets”. Clean encoded-style alt strings sitewide.
  8. Add JSON-LD (organization/local) and basic OG tags on key templates.
  9. Reduce duplicated hero HTML on inner pages or vary primary headings/copy per URL.
